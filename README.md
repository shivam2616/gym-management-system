# gym-management-system

A simple Gym Management System built using Python.  
This project helps manage gym member records and basic operations through a graphical interface.

## 🚀 Features
- Add new gym members
- View member details
- Update member information
- Delete member records
- Simple and user-friendly interface

## 🛠 Technologies Used
- Python
- Tkinter (GUI)
- MySQL / Database (if used)
- VS Code
